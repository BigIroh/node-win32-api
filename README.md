# win32-api
Windows win32 apis for [node-ffi](https://github.com/node-ffi/node-ffi)

## What can I do with this?
Calling win32 native functions come from user32.dll, kernel32.dll by javascript

## Installing
```powershell
npm install --save win32-api
```
